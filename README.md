# 項目名稱

**KACHA! 電影文旅地圖**  
—— 以影視文化驅動大灣區文旅發展的智能服務平台

## 項目簡介

KACHA! 電影文旅地圖是一個影視主題深度文旅服務平台，專注整合香港、內地及大灣區電影拍攝場景資源，以智能技術連接文旅體驗、在地文化與商業服務，為用戶提供一站式電影主題旅行解決方案。

本項目源自我們 [ChinaVis](https://www.chinavis.org/2024/challenge.html) 數據視覺化競賽銅獎作品原型，現已全面商業化升級，具備地圖導航、智能路線規劃、專業拍攝指南、周邊商家推薦、線上預訂等完整功能，並以 AI 自動生成電影文旅計劃為核心亮點，實現影視文藝傳播、在地文化活化與區域經濟可持續發展的多重價值。

## 解決的核心問題

1. **文旅資訊碎片化**  
   電影取景地資料分散、缺乏統一入口，用戶難以快速獲取權威、完整的場景資訊。

2. **行程規劃成本高**  
   影迷、遊客自行規劃電影主題路線耗時、效率低，易走回頭路。

3. **拍攝體驗不足**  
   用戶抵達場景後缺乏專業拍攝指引，難以還原電影經典畫面。

4. **文化解讀淺層化**  
   取景地僅作打卡點，未能連結城市歷史、建築特色與在地文化。

5. **商業流量無法沉澱**  
   電影熱度無法有效轉化為本地消費，周邊商家缺少精準引流渠道。

## 核心功能

### 1. 智能電影文旅路線規劃
- 支持多點電影場景選擇，自動優化遊覽順序
- 整合地圖 API，提供實時路線與步行導航
- 支持按天數、主題、風格自定義行程

### 2. AI 自動生成電影主題旅遊計劃(尚在开发中)
用戶只需輸入：
- 喜愛電影
- 出行天數
- 出發地
- 偏好（文藝 / 美食 / 攝影 / 輕鬆）

即可自動生成：
- 完整每日行程
- 電影場景介紹
- 最佳拍攝時段與角度
- 周邊餐飲、住宿、特色體驗
- 預算與交通建議

### 3. 專業電影拍攝構圖指南
- 電影經典場景還原指導
- 拍攝角度、構圖、光線建議
- 手機 / 相機參數與設備推薦
- 場景氛圍與情緒表達解析

### 4. 在地商家與文旅服務整合
- 電影場點附近餐廳、咖啡廳、文創店、體驗服務
- 商家評分、營業時間、特色、聯絡資訊
- 支持線上預約與服務導覽

### 5. 大灣區文旅資源一體化展示
- 以香港電影數據為核心
- 逐步覆蓋粵港澳大灣區城市
- 推動香港與內地文旅交流融合

## 本地运行

1. Python: `python -m http.server 8000`
2. 在浏览器中打开：`http://localhost:8000/index.html`

## 項目價值

1. **電影文藝價值**
   - 建立專業、系統的電影場景資料庫
   - 延長電影生命週期，從觀影走向體驗
   - 提升公眾影像美學與文藝素養

2. **在地文化價值**
   - 以電影為媒介，講好城市文化故事
   - 活化老街道、老建築、在地文化遺產
   - 促進香港與內地文化交流互鑒

3. **商業與經濟價值**
   - 為本地商家帶來精準電影文旅客流
   - 提高遊客停留時間與消費金額
   - 構建可持續的「影視 + 文旅 + 商業」生態

4. **社會價值**
   - 推動大灣區文旅一體化發展
   - 打造數字文旅創新示範應用
   - 讓文化可體驗、可傳播、可消費、可延續

## 技術架構

- **開發IDE/Agent**：Meituan CatPaw
- **前端**：HTML5 / CSS3 / JavaScript
- **地圖服務**：高德地圖 Web API
- **圖標資源**：Font Awesome
- **數據格式**：JSON 模塊化管理
- **設計**：響應式布局，支持手機、平板、桌面
- **特色**：AI 智能行程生成引擎
